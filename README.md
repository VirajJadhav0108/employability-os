# employability-os
# Employability OS (e-OS)

🚀 AI-powered Internship & Placement Management Platform

Built for Smart India Hackathon 2025.

---

## 📌 Problem Statement

Traditional placement systems rely heavily on emails,
WhatsApp groups, spreadsheets, and manual coordination.

Employability OS centralizes the complete placement workflow
into a single intelligent AI-powered ecosystem.

---

## ✨ Features

- AI-powered internship recommendations
- Student-company skill matching
- Real-time dashboards
- JWT authentication
- Role-based access control
- Placement analytics
- Internship tracking
- Resume/profile management

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Machine Learning
- Scikit-learn
- TensorFlow

---

## 🧠 Recommendation Engine

The recommendation engine uses:
- content-based filtering
- skill matching
- resume keyword analysis
- cosine similarity

to generate personalized internship opportunities for students.
