# Employability OS (e-OS)

AI-powered internship and placement platform built for Smart India Hackathon 2025.

Employability OS (e-OS) is designed to simplify campus recruitment by providing a single platform for students, recruiters, and placement administrators. Instead of managing internships through emails, spreadsheets, and messaging platforms, the system centralizes internship postings, applications, candidate tracking, and placement analytics.

---

## Features

### Student
- Browse internship opportunities
- View personalized internship recommendations
- Apply to internships
- Track application status

### Recruiter
- Post internship opportunities
- View applicants
- Update application status

### Placement Administrator
- Monitor placement activities
- View dashboard analytics
- Manage internships and applications
